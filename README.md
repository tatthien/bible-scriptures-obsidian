## Run

```bash
make run
```

## Usage

When the process finished, there is a directory `scriptures` created. You need to copy and paste it to your Obsidian vault.

Here is my vault:

<img width="1076" alt="Screen Shot 2021-08-11 at 13 30 00" src="https://user-images.githubusercontent.com/72242664/128980385-4a3e2950-e6cd-4c79-9b8e-031664997435.png">
